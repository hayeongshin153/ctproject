💻 컴퓨팅 사고 기반 수업 장소 예측 도식

1️⃣ 문제 분해 (Decomposition)
전체 시간표에서 요일, 교시, 과목명, 특수학급 여부, 이동 수업 여부 정보를 분리해요.
⬇️
2️⃣ 추상화 (Abstraction)
해당 요일의 교시에 해당하는 과목만 남기고
특수학급 여부, 이동 수업 여부(체육·음악·미술·과학)만 남겨서 간단한 정보로 정리해요.
⬇️
3️⃣ 패턴 인식 (Pattern Recognition)
조건설명
색칠된 시간특수학급 수업 (우선 적용)
과목이 체육·미술·음악·과학특별실 수업 (운동장, 미술실 등)
그 외 과목일반 교실 수업
⬇️
4️⃣ 알고리즘 (Algorithm)
if 특수학급이라면 → "특수학급"
else if 이동 과목이라면 → 과목별 장소 적용
else → "일반학급"
⬇️
🏫 출력: 현재 수업 장소는 “OOO”입니다!
